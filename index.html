Feature extraction

How raw data becomes model input

1. Clean raw recordings — trim phone-handling artifacts from start and end

2. Chunk longer recordings into 30-second segments (~3000 samples each)

3. Extract 6 statistical features per chunk from sway magnitude:

  • Mean & Median (central sway tendency)
  • Standard Deviation (sway variability)
  • Skewness & Kurtosis (distribution shape)
  • Path Length (cumulative movement)

4. Classify with SVM (linear kernel)

How it works

The Romberg test measures postural sway

1. Sway magnitude — compute magnitude from accelerometer x, y, z axes to capture total body sway.

2. Six features — mean, median, std, skewness, kurtosis, and path length of the sway signal.

3. Key insight — eyes closed increases postural sway because the brain loses visual feedback for balance correction.
